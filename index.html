DeltaLLM Helm Repository

Add this repository with:

helm repo add deltallm https://deltawi.github.io/deltallm
helm repo update

Quick-start evaluation install with bundled PostgreSQL and Redis:

helm install deltallm deltallm/deltallm \
--version 0.1.20-rc2 \
-f https://deltawi.github.io/deltallm/values-eval-0.1.20-rc2.yaml

Release-matched values overlays:

  • values-eval-0.1.20-rc2.yaml
  • values-production-0.1.20-rc2.yaml
  • values-0.1.20-rc2.yaml

See the main project docs for deployment examples.
