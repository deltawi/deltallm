# Features

DeltaLLM provides enterprise-grade features on top of the unified LLM API.

| Feature | Description |
|---------|-------------|
| [Authentication & SSO](authentication.md) | Session login, master key, SSO with multiple identity providers |
| [Routing & Failover](routing.md) | Load balancing strategies with automatic failover |
| [Caching](caching.md) | Response caching with multiple backends |
| [Guardrails](guardrails.md) | PII detection and prompt injection protection |
| [Rate Limiting](rate-limiting.md) | Hierarchical rate limits at every scope level |
| [Budgets & Spend](budgets.md) | Budget enforcement and cost tracking |
| [Observability](observability.md) | Prometheus metrics, logging, and analytics |
