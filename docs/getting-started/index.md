# Getting Started

This section covers everything you need to get DeltaLLM up and running.

## Choose Your Setup

| Method | Best For | Time |
|--------|----------|------|
| [Installation](installation.md) | Local development, contributing | ~10 min |
| [Quick Start](quickstart.md) | First-time setup walkthrough | ~5 min |
| [Docker](docker.md) | Production-ready deployment | ~5 min |

## Prerequisites

- **Python 3.11+**
- **Node.js 20+** (for the admin UI)
- **PostgreSQL** database
- **Redis** (for rate limiting and caching)
- At least one LLM provider API key (OpenAI, Anthropic, Groq, etc.)
