# Dashboard

The Dashboard page provides an at-a-glance overview of gateway activity.

## Overview Cards

The top section displays summary statistics:

- **Total Spend** — Cumulative cost across all API usage
- **Total Requests** — Number of proxied LLM requests
- **Total Tokens** — Input + output tokens processed
- **Active Models** — Number of configured model deployments

## Daily Spend Chart

A line chart showing spend over the past 30 days, with breakdowns by model.

## Model Usage

A pie chart showing the distribution of requests across model groups, helping identify which models are most used.
